Chapitre 2 - Exercice 3 : La Carte d'identité

Créez une fonction IDCard. Elle prend 3 variables en paramètre. Elles représentent dans l'ordre :

  • Le nom
  • l'age
  • le sexe

de la personne qui va appeler la fonction.

Lorsque la fonction est appelée, une boite de dialogue s'ouvre, dedans une phrase décrit la personne.

Par exemple : "Vous êtes Mr. X et vous avez XXans"
